| 세대 갈등 심화 | 4회 | Rising | 10회+ → Strong |

### 전환 추적 프레임워크

```
분기 초 상태 → 분기 말 상태:
- Emerging → Rising: 성공적 조기 탐지
- Emerging → Stable: 유효한 지속 모니터링
- Emerging → Faded: 일시적 현상 (기록용)
- Missed → Rising: 사후 분석 필요
```
