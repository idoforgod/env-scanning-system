# Report Format

환경스캐닝 일일 보고서 표준 포맷.

## 보고서 구조

### 1. Executive Summary (1페이지)

```markdown
# 환경스캐닝 일일 보고서
**날짜**: 2026년 1월 9일

## 핵심 발견 (Top 3)

### 🔴 [신호 제목 1]
- **카테고리**: Technological
- **중요도**: ★★★★★ (5/5)
- **핵심**: 한 줄 요약

### 🟠 [신호 제목 2]
- **카테고리**: Political
- **중요도**: ★★★★☆ (4/5)
- **핵심**: 한 줄 요약

### 🟡 [신호 제목 3]
- **카테고리**: Economic
- **중요도**: ★★★★☆ (4/5)
- **핵심**: 한 줄 요약

## 오늘의 요약
- 신규 탐지 신호: N건
- STEEPS 분포: S(n) T(n) E(n) E(n) P(n) S(n)
- 주목 영역: [키워드들]
```

### 2. 신규 탐지 신호 (NEW)

STEEP 카테고리별 정리.

```markdown
## 신규 탐지 신호

### Social (사회) - N건
| ID | 제목 | 중요도 | 출처 |
|----|------|--------|------|
| SIG-... | ... | ★★★☆☆ | ... |

#### SIG-2026-0109-001: [제목]
- **설명**: ...
- **잠재적 영향**: ...
- **관련 행위자**: ...
- **출처**: [링크](url)

### Technological (기술) - N건
...

### Economic (경제) - N건
...

### Environmental (환경) - N건
...

### Political (정치) - N건
...
```

### 3. 기존 신호 업데이트

```markdown
## 기존 신호 업데이트

### 상태 변화
| ID | 제목 | 이전 상태 | 현재 상태 | 변화 내용 |
|----|------|----------|----------|----------|
| SIG-... | ... | emerging | developing | 추가 사례 확인 |

### 강화 추세 (↑)
- **SIG-xxx**: [제목] - 추가 증거 확인
  - 변화 내용 설명

### 약화 추세 (↓)
- **SIG-xxx**: [제목] - 관심 감소
  - 변화 내용 설명
```

### 4. 패턴 및 연결고리

```markdown
## 패턴 및 연결고리

### 신호 클러스터
```
[AI 규제 강화]
    ├── SIG-001: EU AI Act 시행
    ├── SIG-023: 미국 AI 행정명령
    └── SIG-045: 중국 AI 거버넌스 지침
```

### 떠오르는 테마
1. **[테마 1]**: 관련 신호들의 공통점과 시사점
2. **[테마 2]**: ...

### 교차 영향 분석
- SIG-001 × SIG-023 → 글로벌 AI 규제 표준화 가속
```

### 5. 전략적 시사점

```markdown
## 전략적 시사점

### 즉각 대응 권고
1. **[권고 1]**: 구체적 행동 제안
   - 관련 신호: SIG-xxx
   - 긴급도: 높음

### 모니터링 강화 영역
| 영역 | 이유 | 관련 신호 |
|------|------|----------|
| AI 규제 | 급변 가능성 | SIG-001, SIG-023 |
| 에너지 전환 | 정책 변화 | SIG-045 |

### 시나리오 업데이트 필요
- 기존 시나리오 X에 대한 재검토 권고
```

### 6. 부록

```markdown
## 부록

### A. 전체 신호 목록
| ID | 카테고리 | 제목 | 중요도 | 상태 |
|----|----------|------|--------|------|
| ... | ... | ... | ... | ... |

### B. 출처 및 참고자료
1. [출처 1](url)
2. [출처 2](url)

### C. 방법론
- 스캐닝 기간: 2026-01-08 ~ 2026-01-09
- 검색 소스: N개
- 중복 제거: N건
- 분석 모델: Claude

### D. 용어 정의
- **약한 신호 (Weak Signal)**: ...
- **STEEP**: ...
```

## 파일 명명 규칙

```
environmental-scan-{YYYY-MM-DD}.md
```

예: `environmental-scan-2026-01-09.md`

## 시각화 가이드

### 중요도 표시
- ★★★★★ (5): 빨간색 강조
- ★★★★☆ (4): 주황색
- ★★★☆☆ (3): 노란색
- ★★☆☆☆ (2): 회색
- ★☆☆☆☆ (1): 연한 회색

### STEEP 아이콘
- 🧑‍🤝‍🧑 Social
- 💻 Technological
- 💰 Economic
- 🌍 Environmental
- 🏛️ Political
