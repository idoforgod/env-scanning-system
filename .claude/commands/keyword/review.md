# 월간 키워드 리뷰

오늘 날짜: {{datetime}}
인자: $ARGUMENTS

---

## 리뷰 개요

STEEPS 키워드의 월간 효과성을 분석하고 개선 권고를 생성합니다.

**갱신 주기**: 매월 말 (또는 수동 실행)

---

## 실행 절차

### Step 1: 대상 월 결정

```
인자가 있으면: $ARGUMENTS (예: 2026-01)
인자가 없으면: 현재 월 자동 계산
```

### Step 2: 월간 메트릭 집계

```bash
# KeywordTracker로 월간 리포트 생성
python3 src/scripts/utils/keyword_tracker.py report {YYYY-MM}
```

### Step 3: 키워드 매니저로 리뷰 수행

```bash
# KeywordManager로 월간 리뷰 실행
python3 src/scripts/utils/keyword_manager.py review {YYYY-MM}
```

### Step 4: 결과 분석

리뷰 결과에서 다음을 확인:

| 항목 | 확인 내용 |
|------|-----------|
| 등급 분포 | A/B/C/D/F 비율 |
| 퇴출 후보 | 3개월 연속 0건 키워드 |
| 상위 성과자 | Top 10 키워드 |
| 개선 필요 | D등급 20% 초과 여부 |

### Step 5: 권고안 생성

```
┌─────────────────────────────────────────────────────────────────┐
│  월간 키워드 리뷰 결과 보고서                                    │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  1. 요약 통계                                                    │
│     - 총 키워드 수: {N}개                                        │
│     - 활성 키워드: {활성 수}개 (수집 1건+)                       │
│     - 비활성 키워드: {비활성 수}개 (수집 0건)                    │
│                                                                  │
│  2. 등급 분포                                                    │
│     - A (우수): {N}개                                            │
│     - B (양호): {N}개                                            │
│     - C (보통): {N}개                                            │
│     - D (저조): {N}개                                            │
│     - F (비효과): {N}개                                          │
│                                                                  │
│  3. 권고 조치                                                    │
│     - 퇴출 후보: [키워드 목록]                                   │
│     - 개선 필요: [키워드 목록]                                   │
│     - 추가 검토: [신규 키워드 후보]                              │
│                                                                  │
│  4. 다음 단계                                                    │
│     - [ ] 퇴출 후보 최종 검토                                    │
│     - [ ] 신규 키워드 추가 여부 결정                             │
│     - [ ] 카테고리별 균형 확인                                   │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### Step 6: 결과 저장

```
저장 위치: data/metrics/monthly-review-{YYYY-MM}.json
```

---

## 사용 예시

```bash
# 현재 월 리뷰
/keyword:review

# 특정 월 리뷰
/keyword:review 2026-01

# 지난 달 리뷰
/keyword:review 2025-12
```

---

## 연관 명령어

- `/keyword:stats` - 키워드 통계 조회
- `/keyword:add` - 키워드 추가
- `/keyword:quarterly` - 분기별 전체 재검토
